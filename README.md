# Study Design Branch

## Purpose

This branch contains the conceptual and methodological design of the project.

It defines:

- Research objectives
- Research questions
- Variable structure
- Data source planning
- Analytical strategy

No data processing or modeling is implemented in this branch.

---

## Role Within the Project

The study-design branch serves as the theoretical and structural foundation for the implementation phase.

Subsequent branches (e.g., `data-collection`, `eda`, `modeling`) will build upon this framework.

---

## Status

Conceptual phase – no empirical results yet.
